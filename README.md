# Mobility System for Blind Pedestrians

Hardware Tools:
  - Raspberry Pi 4 (8GB RAM)
  - Camera module
  - GPS SIM 900 module
  - Bluetooth headphone
  - Power Bank (20,000 mAh)


Software Tools:
  - Python 3
      - PyAudio
      - Playsound
      - Speechrecognition 
  - YOLO v3 Algorithm
  - LabelImg
  - Google Colab
  - OpenCV
  - GPSd
      - OpenStreet Map
